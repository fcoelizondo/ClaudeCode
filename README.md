# ClaudeCode

## Impostor — Field Briefing

A pass-the-phone social deduction game. `index.html` is a single self-contained page — open it directly in a browser, or enable GitHub Pages on this repo to serve it at the repo's Pages URL.

Set the number of agents and impostors, pick a case-file category, then pass the device around: each player holds a button to reveal their word (or their impostor status), then holds another to conceal it before passing it on. Ends in a discussion timer and a full reveal of who was the impostor.
